# Crystal SNMP Library

A lightweight SNMP (Simple Network Management Protocol) library for **Crystal**.
---

## Installation

Add this to your `shard.yml`:

```yaml
dependencies:
  snmp:
    github: lolsonx/crsnmp
```
Then run:
```bash
shards install
```
